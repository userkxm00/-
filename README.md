## 📌 نظرة عامة

**منارة وليد** هي منصة تعليمية رقمية متكاملة مصممة خصيصاً لتدريس المواد الشرعية الإسلامية لطلاب المرحلتين **الإعدادية والثانوية** في مصر.

تتيح المنصة للطلاب الوصول إلى محتوى تعليمي ثري (فيديوهات، مذكرات، اختبارات) بعد تفعيل الاشتراك بكود خاص، مع لوحة تحكم أدمن متكاملة لإدارة كل شيء من مكان واحد.

## ✨ مميزات الطالب

### 🔑 التسجيل والوصول
- إنشاء حساب بالاسم، رقم الهاتف، رقم ولي الأمر، المحافظة، الصف، وكود التفعيل
- التحقق التلقائي من صلاحية الكود ومطابقته للصف الدراسي
- حساب تاريخ انتهاء الاشتراك فور استخدام الكود
- تسجيل دخول آمن عبر Supabase Auth

---

### 📊 لوحة التحكم
- رسالة ترحيب شخصية باسم الطالب
- شريط تقدم الاشتراك مع عدد الأيام المتبقية
- إحصائيات المحتوى المتاح (فيديوهات / مذكرات / اختبارات)
- تتبع التقدم: كم فيديو شاهدت، كم مذكرة قرأت
- آخر المحتوى المضاف لصف الطالب

---

### 🎬 مكتبة الفيديوهات
- فيديوهات مخصصة لصف الطالب فقط
- مشغّل YouTube مدمج داخل المنصة
- إضافة للمفضلة وتسجيل المشاهدة
- فلترة بالمادة والوسوم

---

### 📄 المذكرات والملفات
- عرض وتحميل ملفات PDF المخصصة للصف
- تسجيل الملفات المقروءة تلقائياً

---

### 📝 الاختبارات التفاعلية
- أسئلة متعددة الاختيارات مع تصحيح فوري
- إظهار الإجابة الصحيحة والخاطئة
- حفظ النتيجة وسجل الاختبارات السابقة
- اكتساب نقاط عند الإجابة الصحيحة

---

### 🃏 البطاقات التعليمية (Flashcards)
- نظام تكرار متباعد **Spaced Repetition** للحفظ الذكي
- الطالب يقيّم كل بطاقة (صعب / متوسط / سهل)
- الجدولة التلقائية لموعد المراجعة القادمة

---

### 📅 خطة الدراسة
- جدول أسبوعي تفاعلي (صباح / ظهر / مساء)
- نظام **Streak 🔥**: تتبع الأيام المتواصلة
- عرض المحتوى المقترح لليوم

---

### 🏆 الإنجازات والنقاط
- نظام نقاط مكتسبة من النشاط والاختبارات
- **4 مستويات:** مبتدئ ← متوسط ← متقدم ← نخبة
- **6 شارات قابلة للفتح:**

| الشارة | الشرط |
|--------|-------|
| ⭐ أول خطوة | أول تسجيل دخول |
| 🔥 منتظم | 7 أيام متواصلة بالدراسة |
| 🎬 مشاهد نشيط | مشاهدة 10 فيديوهات |
| ✅ متفوق | 100% في اختبار |
| 🏅 متميز | جمع 500 نقطة |
| 👑 نخبة الطلاب | جمع 1000 نقطة |

---

### 💬 الشات الجماعي
- غرف شات منفصلة لكل صف دراسي
- رسائل فورية عبر **Supabase Realtime**
- مهلة بين الرسائل: **30 ثانية** لمنع الفوضى
- فلتر تلقائي للكلمات المسيئة
- دعم إرسال الصور

---

### 🗣️ منتدى الأسئلة والإجابات

- تعليقات وردود من الطلاب والأستاذ

- تثبيت المنشورات المهمة

---

## 🚀 التثبيت والإعداد

### 1️⃣ متغيرات البيئة

```env
VITE_SUPABASE_URL=https://xxxx.supabase.co
VITE_SUPABASE_ANON_KEY=your_anon_key_here
```

### 2️⃣ تهيئة قاعدة البيانات

افتح **Supabase → SQL Editor** ونفّذ ملف `supabase_schema.sql` كاملاً.

يشمل الملف تلقائياً:
- ✅ إنشاء جميع الجداول (21 جدول)
- ✅ تفعيل RLS وجميع سياسات الحماية
- ✅ إنشاء حساب `admin@walid.com` تلقائياً
- ✅ إعداد Buckets التخزين

### 3️⃣ إنشاء Buckets في Supabase Storage

| Bucket | النوع | الاستخدام |
|--------|-------|-----------|
| `pdfs` | public | ملفات المذكرات |
| `logos` | public | شعار المنصة |
| `broadcasts` | public | ملفات الإشعارات |
| `chat` | public | صور الشات |
| `forum` | public | صور المنتدى |

### 4️⃣ تشغيل المشروع

```bash
pnpm --filter @workspace/manarat-walid dev
```
